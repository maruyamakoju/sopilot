ONE COMMAND EVALUATION
----------------------

Basic usage (JSON output):

    python scripts/sopilot_evaluate_pilot.py \
        --gold demo_videos/manufacturing/oil_change_gold.mp4 \
        --trainee demo_videos/manufacturing/oil_change_trainee_1.mp4 \
        --sop oil_change


PDF output (requires: pip install reportlab):

    python scripts/sopilot_evaluate_pilot.py \
        --gold demo_videos/manufacturing/oil_change_gold.mp4 \
        --trainee demo_videos/manufacturing/oil_change_trainee_1.mp4 \
        --sop oil_change \
        --out report.pdf

================================================================================
OUTPUT INTERPRETATION
================================================================================

Exit Codes:
    0 = PASS (trainee ready for certification)
    1 = FAIL (trainee needs re-training)

Score Grades:
    90-100 = A (Excellent)
    80-89  = B (Good)
    70-79  = C (Acceptable)
    60-69  = D (Needs Improvement)
    0-59   = F (Fail)

Severity Levels:
    CRITICAL = Missing safety step → Immediate re-training required
    HIGH     = Major deviation → Re-training recommended
    MEDIUM   = Quality issue → Coaching recommended
    LOW      = Minor variance → Optional coaching

================================================================================
BATCH PROCESSING
================================================================================

Evaluate multiple trainees:

    #!/bin/bash
    GOLD="demo_videos/manufacturing/oil_change_gold.mp4"
    for trainee in demo_videos/manufacturing/oil_change_trainee_*.mp4; do
        python scripts/sopilot_evaluate_pilot.py \
            --gold "$GOLD" \
            --trainee "$trainee" \
            --sop oil_change \
            --out "reports/$(basename $trainee .mp4).json"
    done


================================================================================
TROUBLESHOOTING
================================================================================

"Gold video not found":
    → Run: python scripts/generate_manufacturing_demo.py

"reportlab not installed" (PDF output):
    → Run: pip install reportlab

Low scores despite correct execution:
    → Adjust threshold with --threshold flag
    → Use --verbose for detailed metrics
