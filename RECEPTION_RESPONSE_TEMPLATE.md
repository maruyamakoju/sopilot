## 使い方（Day 1の作業フロー）

### Step 1: データ受領検知
```bash
# Google Driveから手動ダウンロード、または通知で検知
# → demo_videos/partner/ に配置
```

### Step 2: 一次検品実行
```bash
python scripts/validate_partner_videos.py \
    --input-dir demo_videos/partner \
    --output-format terminal

# 出力を確認して、上記テンプレートの「一次検品結果」に要約
```

### Step 3: 検品結果をもとにテンプレート埋め
- 受領日時を記入
- ファイル数をカウント
- 検品結果（OK/要確認）を転記
- 返却予定日を計算（受領日+7営業日）

### Step 4: 30分以内に返信送信
- 相手に「ちゃんと受け取って確認した」安心感を即座に与える
- 不足があれば**この1回だけ**で追加依頼（後で何度も聞かない）

---

## 重要：このタイミングでやらないこと

❌ 詳細な評価結果（まだ出ていない）
❌ 技術的な問題の詳細（相手は気にしない）
❌ 追加機能の提案（評価が終わってから）

✅ やるべきこと: **「受け取った、確認した、X日に返す、不足があれば今言う」だけ**
