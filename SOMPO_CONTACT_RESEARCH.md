## 🚨 注意事項

### やってはいけないこと
```
❌ 個人メールアドレスを推測して送信
   （例: yamada@sompo-japan.co.jp）
   → スパム扱いされる可能性

❌ 複数の部門に同時送信
   → 社内で混乱を招く

❌ しつこいフォローアップ
   → 悪印象を与える

❌ 誇大広告的な表現
   → 信頼性を損なう
```

### やるべきこと
```
✅ 公式チャネルを使用
✅ 丁寧かつ簡潔な文章
✅ 具体的な数値とエビデンス
✅ リスクゼロの提案（無料PoC）
✅ 適切なフォローアップ間隔（1週間）
```
