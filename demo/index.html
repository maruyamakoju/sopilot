手動SOP審査の構造的限界

  • ✗
    熟練レビュアーが1件の審査に平均25〜40分を要する
    映像スクラビングだけでレビュー時間の40〜60%を消費。500人規模の施設では月間数千時間に達する
  • ✗
    人間レビュアーの逸脱見落とし率: 15〜23%（文献値）
    認知疲労・採点基準のばらつき・主観的判断により、許容できない分散が生じる（Reason, 1990; 産業安全研究所, 2019）
  • ✗
    安全違反は事故発生後まで検知されない
    手動CCTVレビューのカバレッジは映像全体の5%未満。大半の違反は見落とされる
根本原因: 映像データの生成量が人間のレビュー処理能力を超えている。国内製造業の年間労働災害コストは約1.2兆円（厚生労働省, 2023）。
■ 低速 4h映像 × 50名/月 を目視確認 ■ 見落とし 重大逸脱の4件に1件を未検知 ■ コスト高 熟練レビュアーの工数を大量消費 ■ スケール不可 CCTVをリアルタイム監視できない ──────────────────────────────────── 結果: 事故・法的責任・再教育コスト

VigilPilot — リアルタイム違反検知

自然言語でルールを定義
POST /vigil/sessions { "name": "Construction Site Safety", "rules": [ "作業者がヘルメットを着用していない場合に検出", "安全ベルトなしで高所作業している人を検出", "立入禁止エリアへの侵入を検出" ], "severity_threshold": "warning", "backend": "claude" }
ルールはゼロショットVLMプロンプト — MLトレーニング不要。 ルール変更はリアルタイムに反映（再デプロイ不要）。日本語ルール記述をネイティブサポート。
違反イベントレスポンス
// GET /vigil/sessions/{id}/events { "event_id": 42, "timestamp_sec": 183.5, "violations": [{ "rule": "ヘルメット未着用", "severity": "critical", "confidence": 0.94, "bboxes": [[234, 112, 498, 287]] }], "frame_url": "/vigil/events/42/frame?annotate=true" }
