# SOPilot 撮影ガイド（1ページ版）

## 目的
- 現場動画を「採点可能な品質」で安定して収集する。

## 推奨撮影
- 顔より手元と対象物を優先
- 画角は固定（スマホは三脚推奨）
- 開始前に対象物全体を2秒写す
- 作業終了後に結果状態を2秒写す
- 逆光/暗所を避ける

## NG例
- 作業中に大きくパン/ズームする
- 人の出入りが多く主対象が隠れる
- 手元がフレーム外に出続ける
- 音声だけで手順を補う前提（映像に残らない）

## プライバシー配慮
- 顔・名札・個人情報ラベルは映さない
- 必要時はマスク対象を事前申告（名札位置など）

## ファイル運用
- `task_id` ごとにフォルダ分離
- ファイル名: `YYYYMMDD_site_task_operator.mp4`
- 失敗例を消さずに残す（改善学習に必須）
