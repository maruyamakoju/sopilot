# SOPilot 有償PoCパッケージ（テンプレート）

## 1. 目的
- 現場のSOP評価を、目視レビューから半自動評価へ置換し、教育/監査工数を削減する。

## 2. スコープ固定（初回提案の標準）
- `task_id`: 1つのみ
- Gold動画: 1本
- Trainee/Audit動画: 10〜30本
- 評価観点: `missing` / `swap` / `deviation` の3種
- 期間: 4〜6週間

## 3. 納品物（営業時に提示する最小セット）
- Before/Afterレポート（評価工数、監査レポート作成工数）
- 逸脱検知サンプル3〜5ケース（タイムコード付き）
- 監査提出用PDF一式（`/score/{id}/report.pdf`）
- 次フェーズ改善提案（運用・撮影・閾値調整）

## 4. 受け入れ基準（Done）
- 30分動画を落ちずに処理できる
- Gold/Trainee比較で逸脱箇所へジャンプできる
- 採点結果をJSON/PDFで出力できる
- ジョブ単位で追跡できる（誰が/いつ/何を/結果）

## 5. 体制
- 顧客側: 現場責任者1名、教育担当1名、情シス窓口1名
- 提供側: PoC責任者1名（単独運用想定）

## 6. 価格の型（例）
- 初期PoC費: `XXX万円`（上記スコープ内）
- 追加動画解析: `X万円 / 10本`
- 次フェーズ（月額運用）: `X万円 / 拠点`

## 7. 責任範囲
- 提供側: システム提供、採点ロジック設定、レポート出力
- 顧客側: 撮影、アップロード、現場ラベル確認、運用判断
