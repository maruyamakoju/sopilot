# SOPilot 有償PoC 固定パッケージ（契約用）

## 1. スコープ（固定）
- 対象タスク: `task_id` 1つ
- Gold動画: 1本
- Trainee/Audit動画: 10-30本
- 期間: 4-6週間
- 評価観点: `missing` / `swap` / `deviation`

## 2. 納品物（固定）
- Before/Afterレポート（評価工数、監査レポート作成工数）
- 逸脱検知サンプル 3-5ケース（タイムコード付き）
- 採点JSON + PDF監査レポート
- 次フェーズ改善提案（運用/撮影/閾値設定）

## 3. 受け入れ基準（Done）
- 30分動画を安定処理できる
- 逸脱箇所へUIからジャンプできる
- PDF監査レポートを出力できる
- `job_id` で監査トレイルを追跡できる

## 4. 前提条件（顧客側）
- 撮影協力者と評価責任者を各1名以上アサイン
- 撮影ガイドに沿って動画を収集
- 情シス窓口を1名設定

## 5. 除外事項（初回PoCでは実施しない）
- 複数 `task_id` の同時評価
- 業務システム連携（PLC/POS/MES連携）
- 高度なマスキング（PoC中は簡易運用）
- 本番SLA/24時間監視

## 6. 価格レンジ（例）
- PoC一式: 120-300万円（固定スコープ）
- 追加動画: 10本ごとに追加見積
- 本運用移行時: 拠点年額ライセンスへ切替

## 7. 成果判定KPI（例）
- 1本あたり評価時間: 20分 -> 2-5分
- 重要逸脱の見逃し率: PoC開始時より改善
- 監査レポート作成時間: 50%以上短縮
